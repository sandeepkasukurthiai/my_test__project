# FastAPI Student Project

This is a starter FastAPI project structure for students.

## Structure
- `app/` - FastAPI application code
- `tests/` - Unit tests
- `docs/` - Documentation

## Quickstart
1. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
2. Run the app:
   ```powershell
   uvicorn app.main:app --reload
   ```
3. Run tests:
   ```powershell
   pytest
   ```
